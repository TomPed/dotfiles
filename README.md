# My [.]files

## Getting the [.]files
1. Run `cd ~`
2. Run `git clone git@github.com:TomPed/dotfiles.git`

### Installing the [.]files
1. Run `cd ~/dotfiles`
2. Run `./install.sh` to install these [.]files into your home directory

### Updating the [.]files
1. Run `cd ~/dotfiles`
2. Run `./update.sh` to update the these [.]files from your [.]files
